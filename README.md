# flask-demo

This is a flask demo.

# Commands

0. Set up environment

```bash
vituralenv -p python3 env
. env/bin/activate
pip install flask
```
1. Activate environment

```bash
vituralenv -p python3 env
. env/bin/activate
```

2. Deactivate environment

```bash
deactivate
```

3. Run the app

```bash
export FLASK_APP=hello_world.py
flask run
```
